# 💻 Prácticas de Programación en Java

Este repositorio contiene una selección de mis mejores ejercicios y retos técnicos realizados en el módulo de **Programación** de 1º de DAM.

## 🏆 Proyecto Destacado: Examen 2ª Evaluación
He decidido incluir el código correspondiente al examen de la segunda evaluación como muestra de mis capacidades actuales en Java. En este ejercicio se aplican conceptos avanzados de:

- **Lógica de Control:** Uso complejo de bucles y condicionales.
- **Gestión de Datos:** Manipulación de estructuras (Arrays/Listas) y validación de entradas.
- **Eficiencia:** Resolución de problemas bajo tiempo limitado, aplicando buenas prácticas de programación.

## 🛠️ Stack Técnico
- **Lenguaje:** Java 17+
- **Paradigma:** Programación imperativa y modular.
- **Entorno:** Desarrollo realizado en IDE profesional (IntelliJ).
